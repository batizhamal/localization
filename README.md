# localization

## Что есть на данный момент:

- Функция загружать файлы в любом порядке (при обновлении страницы не пропадают данные, записываются в localStorage при загрузке файлов)
- Первый загруженный файл является главным (primary)
- и последующие загруженные файлы отталкиваются от первого
- Parsing of nested objects (берет primary файл и вытаскивает ключи)
- Нахождение дельты между primary файлом и остальными
- Функция редактировать json файлы (при обновлении не пропадают изменения, записываются в localStorage)
- Функция скачивать json файлы
- Функция очистить localStorage
