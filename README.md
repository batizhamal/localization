# localization

## Что есть на данный момент:

- Можно загрузить ru.json (при обновлении страницы не пропадают данные, записываются в localStorage при загрузке файлов)
- Работает parsing of nested objects (берет ru.json и вытаскивает ключи)
- ВРЕМЕННО НЕЛЬЗЯ (Можно загрузить kz.json, показывается дельта пустыми полями (при обновлении страницы не пропадают данные, записываются в localStorage при загрузке файлов))
- Можно редактировать ru.json (при обновлении не пропадают изменения, записываются в localStorage)
- Можно скачать ru.json
- Можно редактировать kz.json (при обновлении не пропадают изменения, записываются в localStorage)
- Можно скачать kz.json
